AVL Tree Insertion
  1. Create an empty AVL tree.
  2. Insert the following values into the tree: 10, 20, 30, 40, 50.
  3. Perform an in-order traversal of the tree and store the elements in a list.
  4. Compare the elements in the list with the expected order: [10, 20, 30, 40, 50].
  5. Validate that the tree is balanced after all insertions are completed.

Expected outcome:

  • The AVL tree should maintain its balance property after each insertion and removal operation.
  • The elements retrieved through in-order traversal should match the expected order.
